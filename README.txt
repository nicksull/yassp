Social share buttons for Facebook, Twitter, Google+, Pinterest, LinkedIn and Whatsapp.

== Description ==

This plugin will produce social share buttons for Facebook, Twitter, Google+, Pinterest, LinkedIn and Whatsapp. Allows the user to configure the order of share buttons, choose button sizes & select colour of icons. The buttons can be placed below the post title, floating on the left, after the post content or inside the featured image. Also included is a shortcode to allow the user to include the social share buttons inside a post.

== Installation ==

via Wordpress
1. Go to your admin area and select Plugins → Add New from the menu.
2. Click on Upload Plugin button
3. Click Choose Now button and select the zip file `toptal-social-share-button.zip`
4. Click install.
5. Click activate to activate plugin.
6. Navigate to Settings → Social Share Buttons

via FTP
1. Upload `toptal-social-share-buttons` folder and contents to the `/wp-content/plugins/` directory using a FTP program
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Settings → Social Share Buttons
